# 项目文件结构

```js
    --> src

        --> assets // 存放静态资源的地方
        --> components // view层
        --> eventBus // bus时间总线
            --> eventBus.js

        --> https // axios封装
            --> filter.js // 拦截器
            --> https.js // 请求操作
            --> url.js // 请求地址

        --> router // 路由
            --> index.js

        --> store  // vuex状态管理
            --> acitons.js
            --> getters.js
            --> index.js
            --> mutation-types.js
            --> mutations.js
            --> state.js

        --> App.vue
        --> main.js

    --> static
        --> projectConfig // 全局
            --> projectConfig.js
        --> tools 工具
            --> rem.js

```
